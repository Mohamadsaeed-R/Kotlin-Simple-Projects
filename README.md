# Kotlin Simple Projects

This repository contains small Kotlin programs and exercises that I build while learning the Kotlin programming language.

The goal of this repository is to practice core Kotlin concepts through simple examples and mini-projects.

## Topics Covered

Examples in this repository may include:

- Variables and data types
- Control flow (if, when, loops)
- Functions
- Collections (List, Set, Map)
- Object-Oriented Programming in Kotlin
- Basic problem solving
- Small console-based programs

## Purpose

This repository serves as:

- A personal learning journal for Kotlin
- A collection of simple practice programs
- A reference for basic Kotlin concepts

## Future Plans

As I progress, I plan to add:

- More advanced Kotlin examples
- Algorithm practice
- Android projects built with Kotlin
